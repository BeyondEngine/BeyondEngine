这是超越引擎正式公布的第一个版本。我们想告知大家以下几点：

1. 引擎基本介绍请参阅：http://blog.csdn.net/beyondlwm/article/details/40427457
视频请参阅：http://v.youku.com/v_show/id_XODEyMDU5MDg0.html

2. 由于人手和资金等等原因，引擎整体完成度并不高，代码质量也难免出现参差不齐的情况，bug以及崩溃也不在少数，我们在日后会努力和各位一起改进，还请轻喷，见谅。

3. 由于团队着重在项目开发之上，目前引擎没有专门的人员维护，所以目前引擎研发基本由项目需求来驱动。希望大家积极反馈意见和建议，我们会尽可能的及时处理。

4. 我们开源的目的主要是为了促进各位同仁的交流，我们的开发团队在成长过程中也同样受到了许多开源项目的帮助，所以我们希望能秉承这份精神，将自己的一些想法和思路公布出来，以供大家参考，共同提高。

5. 我们认为超越引擎还处于一个刚刚起步的阶段，而且，证明一个引擎最好的方式就是它所产出的游戏项目质量，所以我们未来很长一段时间会将重心致力于项目开发。请大家关注《星际奇兵》，我们预计在2015年4月左右上线。

6. 关于文档，教学，sample的缺失，还请大家多包涵。如果大家有疑问可以到群或者贴吧提出来，我们会尽量抽时间来解答。

7. 我们团队位于成都，目前成员有7人。我们是一群有创业激情和游戏梦想的开发者，都曾经在大公司担任过管理岗位，生活可谓无忧。但是我们为了共同的梦想以及实现自身的抱负，毅然放弃了安逸的环境，投身于创业之中。我们真诚邀请有共同志向和理念的朋友加入我们，把工作转变为事业，把事业转变为理想。
